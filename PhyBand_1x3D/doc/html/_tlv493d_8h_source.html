73  /* sets the data access mode for TLE493D
74  * Tle493d is initially in POWERDOWNMODE
75  * use POWERDOWNMODE for rare and infrequent measurements
76  * Tle493d will automatically switch to MASTERCONTROLLEDMODE for one measurement if on a readout
77  * measurements are quite slow in this mode. The power consumption is very low between measurements.
78  * use MASTERCONTROLLEDMODE for low measurement frequencies where results do not have to be up-to-date
79  * In this mode a new measurement starts directly after the last result has been read out.
80  * use LOWPOWERMODE and ULTRALOWPOWERMODE for continuous measurements
81  * each readout returns the latest measurement results
82  * use FASTMODE for for continuous measurements on high frequencies
83  * measurement time might be higher than the time necessary for I2C-readouts in this mode.
84  * Note: Thus, this mode requires a non-standard 1MHz I2C clock to be used to read the data fast enough.
